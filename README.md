# 42_libft

Le projet consiste à recoder un ensemble de fonctions en C afin de se créer sa propre bibliothèaque. 
Les seules fonctions autorisées sont : malloc, free et write.

Fonctions comprises dans la libc : 
- ft_isalpha
- ft_isdigit
- ft_isalnum
- ft_isascii
- ft_isprint
- ft_strlen
- ft_memcpy
- ft_memmove
- ft_srlcpy
- ft_strlcat
- ft_toupper
- ft_tolower
- ft_strchr
- ft_strrchr
- ft_strncmp
- ft_memchr
- ft_memcmp
- ft_strnstr
- ft_atoi
- ft_calloc 
- ft_strdup

Fonctions non comprises dans la libc :
- ft_substr
- ft_strjoin
- ft_strtrim
- ft_split
- ft_strmapi
- ft_striteri
- ft_putchar_fd
- ft_putstr_fd
- ft_putendl_fd
- ft_putnbr_fd

Fonctions nécessaires à la gestion de listes chaînées :
- ft_lstnew
- ft_lstadd_front 
- ft_lstsize
- ft_lstlast
- ft_lstadd_back
- ft_lstdelone
- ft_lstclear
- ft_lstiter
- ft_lstmap
